| `RecentContentSection` fetches all MCQs without class filter | MEDIUM | Home page recent content |

## Implementation Plan

### 1. Create `ClassContextBanner` Component

**Create: `src/components/shared/ClassContextBanner.tsx`**

A subtle, persistent banner that shows the user's current class context. Placed at the top of content pages (below header, above content).

```tsx
// Visual: 📘 বর্তমান শ্রেণি: SSC ▾  (clickable to switch)
// Only renders when learningMode === 'CLASS_BASED'
// Clicking opens a small popover/dropdown to switch class
// Compact design: single line, muted styling, doesn't dominate the page
```

Design:
- Single-line pill: `<GraduationCap icon> বর্তমান শ্রেণি: {className} <ChevronDown>`
- Background: `bg-emerald-50/50 dark:bg-emerald-950/20` with bottom border
- Positioned: sticky below header (or inline at top of page content)
- On click: opens dropdown with all class options (same as header dropdown)
- Only visible when `isClassBased && isAuthenticated`

### 6. Home Page — RecentContentSection Class Filter

**Modify: `src/components/home/RecentContentSection.tsx`**

The `RecentContentSection` fetches `/api/mcq?limit=10` without class filtering. When in CLASS_BASED mode, it should either:
- Pass `classLevel` to the API (if the MCQ API supports it), or
- Hide this section entirely in CLASS_BASED mode (since it shows random MCQs from all classes)

Recommended: Pass `classLevel` param when in CLASS_BASED mode.

### 7. SubjectExplorerSection — Respect Learning Preference

**Modify: `src/components/home/SubjectExplorerSection.tsx`**

When in CLASS_BASED mode, auto-select the user's class tab instead of defaulting to `class-6`. This makes the subject explorer feel personalized.

```tsx
const { classLevel, learningMode } = useLearningPreference()
const [selectedClass, setSelectedClass] = useState<string>(CLASS_TABS[0])

// Auto-select user's class on mount
useEffect(() => {
  if (learningMode === 'CLASS_BASED' && classLevel) {
    setSelectedClass(classLevel)
  }
}, [learningMode, classLevel])
```
